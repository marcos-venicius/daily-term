# Daily Term

Manage your daily tasks using your terminal with **vim motions**

## Modes

- NORMAL
- COMMAND

## NORMAL mode keybindings

- <kbd>q</kbd> quit
- <kbd>:</kbd> enter `COMMAND` mode
- <kbd>Esc</kbd> clear error

## COMMAND mode commands

- `q` quit
- `quit` quit
- <kbd>Esc<kbd> go back to `NORMAL` mode
